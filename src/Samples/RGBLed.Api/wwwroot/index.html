硬件信息

GPIO 引脚配置:
• 红色LED: GPIO 18 (PWM)
• 绿色LED: GPIO 19 (PWM)
• 蓝色LED: GPIO 20 (PWM)
• 按钮1 (颜色切换): GPIO 2
• 按钮2 (效果切换): GPIO 3
• 按钮3 (亮度调节): GPIO 4

按钮功能:
• 按钮1: 循环切换预设颜色
• 按钮2: 循环切换效果模式
• 按钮3: 循环调节亮度 (25%, 50%, 75%, 100%)
