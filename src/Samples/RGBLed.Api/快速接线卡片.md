### 🔌 第二步：按表接线
| 设备 | 连接到树莓派 | 引脚号 |
|------|-------------|--------|
| 🔴 红色LED | GPIO18 | **12号脚** |
| 🟢 绿色LED | GPIO19 | **35号脚** |
| 🔵 蓝色LED | GPIO20 | **38号脚** |
| ⚫ LED地线 | GND | **6号脚** |
| 🟡 按钮1 | GPIO2 | **3号脚** |
| 🟡 按钮2 | GPIO3 | **5号脚** |
| 🟡 按钮3 | GPIO4 | **7号脚** |
| ⚫ 按钮地线 | GND | **6号脚** |

### ⚡ 第三步：功能测试
- **按钮1**: 切换颜色 (红→绿→蓝→黄→青→紫→白→橙→紫红)
- **按钮2**: 切换效果 (关闭→静态→闪烁→呼吸→彩虹)  
- **按钮3**: 调节亮度 (25%→50%→75%→100%)
